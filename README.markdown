TapesTwitter
============

#### A Tapestry 5 demo application ####


TapesTwitter is a "Twitter-like" demo application build with Tapestry 5.

Build around :

* Tapestry 5
* Spring (IoC and Spring Security)
* JPA (with Hibernate implementation)
* HSQL

Tested with :

* Firefox 3.6
* Chrome 10
* Safari 5.0.4
* IE 8
